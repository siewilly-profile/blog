# f313. 2. 人口遷移
## 解題
我們先釐清已知條件：
- 每次遷移人口會是$城市人數/k$
- 當成是人口為-1時，這個地方不增加人也不減少人

那我們可以用最簡的資料結構去處理這個問題，我們看到move函數，這個函數會開一個新的陣列(plus)，這個plus會記錄每個城市位置的城市人口遷移情況，對於每個四周有城市的地方輸出自己的人(moves就是用來檢查自周城市的)，並且在被輸出的城市加上輸入人數，最後合併arr & plus，這樣我們可以的算出經過一晚的遷移城市人口的數量

再利用遍歷去看有城市的地方最多的人數跟最少的人數
## AC Code
### python
```python
from sys import stdin
r,c,k,m=map(int,stdin.readline().strip().split())
row=[list(map(int,stdin.readline().strip().split())) for _ in range(r)]
moves=[[0,1],[0,-1],[1,0],[-1,0]]
def move(arr):
    plus=[[0]*c for _ in range(r)]
    for i in range(r):
        for j in range(c):
            if arr[i][j]!=-1:
                for dx,dy in moves:
                    nx=i+dx
                    ny=j+dy
                    if 0<=nx<r and 0<=ny<c and arr[nx][ny]!=-1:
                        plus[i][j]-=(arr[i][j]//k)
                        plus[nx][ny]+=(arr[i][j]//k)
    for i in range(r):
        for j in range(c):
            plus[i][j]=plus[i][j]+arr[i][j]
    return plus
for _ in range(m):
    row=move(row)
max_num=float('-inf')
min_num=float('inf')
for i in range(r):
    for j in range(c):
        if row[i][j]!=-1:
            max_num=max(max_num,row[i][j])
            min_num=min(min_num,row[i][j])
print(min_num)
print(max_num)
```
